# Drupal Canvas: Advanced Site Building

A reveal.js presentation on using Drupal Canvas for advanced site building.

## Running the Presentation

```bash
npm install
npm start
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

Slides are authored in [`slides.md`](slides.md) — edit that file to update the presentation. Slides are separated by `---`.
